# namer_app
Aplicación móvil desarrollada en Flutter para administrar una papelería escolar.

## ✨ Funciones

- Catálogo de productos
- Carrito de compras
- Precios de productos
- Diseño moderno con tarjetas
- Contacto por WhatsApp
- Manejo de estado con Provider
- Selección de imágenes
- Interfaz responsiva

---

## 🛠 Tecnologías Utilizadas

- Flutter
- Dart
- Provider
- Google Fonts
- Image Picker
- URL Launcher
